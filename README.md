# Playbook de Vendas

Frontend público do Playbook de Vendas/Cockpit de Calls Consultivas.

## Publicação

Este diretório pode ser enviado para o GitHub Pages. O arquivo principal é `index.html`.

## Importante

Não coloque neste repositório:

- Apps Script do Google;
- backups reais;
- planilhas reais com dados de clientes;
- tokens, senhas ou chaves de acesso.

```text
index.html
styles/main.css
scripts/playbook-content.js
scripts/app.js
assets/
docs/
```
